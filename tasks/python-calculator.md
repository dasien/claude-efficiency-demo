Create a Python command-line calculator application with the following requirements:

1. **Core functionality**: Support addition (+), subtraction (-), multiplication (*), and division (/) operations
2. **Input handling**: Accept input in the format `<number> <operator> <number>` (e.g., `5 + 3`)
3. **Input validation**: Handle invalid inputs gracefully with clear error messages
4. **Division by zero**: Handle division by zero with an appropriate error message
5. **REPL loop**: Run in a read-eval-print loop until the user types `quit` or `exit`
6. **Decimal support**: Support both integer and floating-point numbers
7. **Testing**: Write pytest tests covering all operations, edge cases, and error handling
8. **Project structure**: Create a clean project structure with `calculator.py` for the main code and `test_calculator.py` for tests
9. **Requirements file**: Include a `requirements.txt` with pytest as a dependency

After creating the code, run the tests to verify they pass. Use `python3` (not `python`) for all commands.
